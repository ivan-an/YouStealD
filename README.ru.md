# YouStealD

**YouStealD** — это графический загрузчик и монитор стримов для видео, плейлистов, каналов и прямых эфиров YouTube на базе Qt.

Он позволяет скачивать отдельные видео, плейлисты или целые каналы, а также непрерывно отслеживать каналы на предмет новых загрузок или прямых эфиров и автоматически скачивать их.

Поддерживается аутентификация через файлы `cookies.txt`, экспортированные из браузеров.

Под капотом YouStealD использует **yt-dlp** и **FFmpeg** для надёжного извлечения и обработки медиа, с опциональной интеграцией **aria2** для ускоренной многопоточной загрузки и поддержкой **прокси HTTP/HTTPS/SOCKS**.

## ✨ Основные возможности

- **Скачивание видео / плейлистов / каналов** — выбор формата, разрешения и контейнера (MP4, WebM или только аудио).
- **Запись прямых эфиров** — автоматическая запись стримов в момент их начала.
- **Мониторинг стримов** — следите за каналом и автоматически скачивайте новые видео или прямые эфиры.
- **Авторизованные загрузки** — поддержка файлов `cookies.txt`, экспортированных из браузеров.
- **Высокоскоростная загрузка** — опциональная интеграция **aria2** с поддержкой до **16 соединений** для ускоренного скачивания.
- **Поддержка прокси** — загрузка через **прокси HTTP, HTTPS или SOCKS**.
- **Контроль скорости скачивания** — опциональное ограничение скорости загрузки.
- **Пользовательский User-Agent** — возможность переопределить стандартный User-Agent для запросов.
- **Умное автообновление yt-dlp** — автоматическое обновление yt-dlp при необходимости, с избеганием лишних обновлений при отмене или ошибках сети.

### Пресеты форматов

- **MP4**
  - best, 4K, 1440p, 1080p, 720p, 480p, 360p, 240p, 144p
- **WebM**
  - аналогичные уровни качества
- **Только аудио**
  - MP3, M4A, WAV, OGG

### Оптимизация производительности

- Размер буфера 32 МиБ
- 4 одновременных фрагмента (по умолчанию), до 16 с aria2c
- aria2c: 8 соединений (с ограничением скорости) или 16 соединений (без ограничений)
- опциональный `--no-warnings` для чистого лога
- улучшенная обработка временных файлов

## ⚙️ Компоненты среды выполнения

YouStealD использует несколько внешних инструментов для извлечения и обработки медиа:

- **yt-dlp** — движок извлечения медиа
- **FFmpeg** — обработка и склейка медиа
- **Deno** — JavaScript-рантайм, используемый yt-dlp для извлечения с YouTube
- **aria2c** *(опционально)* — многопоточный загрузчик для ускорения скачивания

Портативный релиз включает все необходимые компоненты среды выполнения.
