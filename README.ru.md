# YouStealD

**YouStealD** — это графический загрузчик и монитор стримов для YouTube на базе Qt.

Он позволяет скачивать отдельные видео, плейлисты или целые каналы, а также непрерывно отслеживать каналы на предмет новых загрузок или прямых эфиров и автоматически скачивать их.

Поддерживается аутентификация через куки браузера или файл `cookies.txt`.

Под капотом YouStealD использует **yt-dlp** и **FFmpeg** для надёжного извлечения и обработки медиа, с опциональной интеграцией **aria2** для ускоренной многопоточной загрузки и поддержкой **прокси HTTP/HTTPS/SOCKS**.

## ✨ Основные возможности

- **Скачивание видео / плейлистов / каналов** — выбор формата, разрешения и контейнера (MP4, WebM, только аудио).
- **Мониторинг стримов** — следите за каналом и автоматически скачивайте новые видео или прямые эфиры.
- **Авторизованные загрузки** — используйте куки из Chrome, Firefox, Edge (или любого браузера на основе Chromium: Zen, Brave, Opera) или предоставьте файл `cookies.txt`.
- **Высокоскоростная загрузка** — опциональная интеграция **aria2** для многопоточного скачивания.
- **Поддержка прокси** — загрузка через **прокси HTTP, HTTPS или SOCKS**.
- **Пресеты форматов**
  - MP4 (лучшее, 1080p, 720p)
  - WebM (лучшее)
  - Только аудио (MP3)
- **Оптимизация производительности**
  - Буфер 1 МиБ
  - 4 одновременных фрагмента
  - опциональный `--no-warnings` для чистого лога
- **JavaScript-рантайм** — включает `deno.exe` (или использует системный Deno), чтобы извлечение с YouTube работало без дополнительной настройки.
- **Современный интерфейс на Qt** — анимированные иконки, сворачиваемая панель лога, адаптивные цвета под тему и уведомления в стиле системного трея.
- **Кроссплатформенность** — собирается на Windows (MinGW/MSVC), Linux и macOS из одних и тех же исходников.

## 🛠️ Сборка и запуск (Qt 6+)

### Требования

| Компонент | Требование |
|-----------|------------|
| Qt | Qt 6 (Core, Gui, Widgets, Network, Xml) |
| Компилятор | MSVC, MinGW‑w64 или clang |
| Инструменты | `yt-dlp.exe`, `ffmpeg.exe`, `deno.exe` |
| Опционально | `aria2c` (для опции `--downloader`) |

### Шаги сборки

```bash
# Клонируйте репозиторий
git clone https://github.com/ivan-an/YouStealD.git
cd YouStealD

# Сгенерируйте Makefile
qmake youtubed.pro

# Соберите проект
mingw32-make
